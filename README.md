## 🚀 Featured Projects

### 🔍 Multimodal Image Retrieval (CLIP + SimCLR)
> Research project combining contrastive vision-language learning (CLIP) and self-supervised representation learning (SimCLR) into a dual-loss training pipeline. Supports both text-to-image and image-to-image retrieval at scale using FAISS similarity search.

**Stack:** `Python` `PyTorch` `FAISS` `ResNet50` `HuggingFace` `MS-COCO` `Google Colab`

---

### 🎬 Netflix-Style Video Streaming App
> Full-stack streaming platform featuring HTTP range request streaming, HLS conversion via ffmpeg, NextAuth session management, and a TypeScript/Node.js video upload pipeline backed by MongoDB metadata storage.

**Stack:** `Next.js` `TypeScript` `Node.js` `MongoDB` `Multer` `ffmpeg` `NextAuth`

---

### 🤖 Jenkins AI Chatbot — GSoC Proposal
> Proposed an AI-powered documentation assistant for Jenkins using a RAG (Retrieval-Augmented Generation) pipeline. Covers vector database indexing, semantic search, and REST API integration.

**Stack:** `Python` `LangChain` `Vector DB` `REST API`

---

### 🌐 OWASP Nettacker WebUI Redesign — GSoC Proposal
> Proposed a full WebUI redesign for OWASP Nettacker including a CISA KEV module integration, visual workflow engine, improved test coverage, and a modernized frontend.

**Stack:** `Python` `JavaScript` `REST API` `OWASP`

---

### 📚 Library Management System
> Java-based library management system implementing full CRUD operations with a clean object-oriented design.

**Stack:** `Java`
